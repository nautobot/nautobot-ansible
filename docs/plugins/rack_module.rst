.. Title

networktocode.nautobot.rack -- Create, update or delete racks within Nautobot
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `networktocode.nautobot collection <https://galaxy.ansible.com/networktocode/nautobot>`_ (version 1.0.4).

    To install it use: :code:`ansible-galaxy collection install networktocode.nautobot`.

    To use it in a playbook, specify: :code:`networktocode.nautobot.rack`.

.. version_added

.. versionadded:: 1.0.0 of networktocode.nautobot

   :depth: 1

Synopsis

.. Description

- Creates, updates or removes racks from Nautobot

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: "Test Nautobot modules"
      connection: local
      hosts: localhost
      gather_facts: False

      tasks:
        - name: Create rack within Nautobot with only required information
          networktocode.nautobot.rack:
            url: http://nautobot.local
            token: thisIsMyToken
            data:
              name: Test rack
              site: Test Site
              status: active
            state: present

        - name: Delete rack within nautobot
          networktocode.nautobot.rack:
            url: http://nautobot.local
            token: thisIsMyToken
            data:
              name: Test Rack
            state: absent
